Problemas de
Covarianza

Cálculo de esperanza conjunta y varianza en sistemas tecnológicos.

🧠 Problema 1 — Pipeline de Machine Learning

En el entrenamiento de una red neuronal, el tiempo de procesamiento en milisegundos se divide en dos variables aleatorias continuas:

  • $X$ = Tiempo de preparación de tensores en CPU.
  • $Y$ = Tiempo de transferencia y cálculo en GPU.

Ambas variables tienen una función de densidad de probabilidad conjunta dada por:

$$f(x,y) = x + y \quad \text{para } 0 \le x \le 1, \ 0 \le y \le 1$$

Se pide: Calcular la covarianza $Cov(X,Y)$ para determinar si existe una relación lineal entre los tiempos de CPU y GPU.

1

Esperanza de X e Y

Primero, calculamos $E[X]$. Por simetría de la función conjunta, $E[Y]$ será igual.

$$E[X] = \int_0^1 \int_0^1 x(x + y) \, dy \, dx$$ $$E[X] = \int_0^1 \left[ x^2y + x\frac{y^2}{2} \right]_0^1 dx$$ $$E[X] = \int_0^1 \left( x^2 + \frac{x}{2} \right) dx = \left[ \frac{x^3}{3} + \frac{x^2}{4} \right]_0^1 = \frac{7}{12}$$
2

Esperanza del producto E[XY]

Calculamos el valor esperado conjunto:

$$E[XY] = \int_0^1 \int_0^1 xy(x + y) \, dy \, dx$$ $$E[XY] = \int_0^1 \int_0^1 (x^2y + xy^2) \, dy \, dx = \frac{1}{3}$$
3

Fórmula de la Covarianza

Aplicamos el teorema: $Cov(X,Y) = E[XY] - E[X]E[Y]$

$$Cov(X,Y) = \frac{1}{3} - \left(\frac{7}{12}\right)\left(\frac{7}{12}\right)$$ $$Cov(X,Y) = \frac{48}{144} - \frac{49}{144} = -\frac{1}{144}$$
Resultado Problema 1
$$Cov(X,Y) = -0.0069$$
💡 Interpretación: Existe una covarianza negativa muy ligera. Esto indica que cuando el tiempo de la CPU es ligeramente más alto, la GPU tiende a procesar marginalmente más rápido, compensando microscópicamente el tiempo total del pipeline.

☁️ Problema 2 — Latencia en Azure

Una aplicación alojada en Microsoft Azure tiene un tiempo de respuesta total $T$ que es la suma del tiempo de cómputo en la Máquina Virtual ($X$) y el tiempo de consulta en la base de datos SQL ($Y$). Entonces, $T = X + Y$.

  • Varianza del servidor: $Var(X) = 4 \text{ ms}^2$
  • Varianza de la BD: $Var(Y) = 9 \text{ ms}^2$
  • Covarianza bajo estrés: $Cov(X,Y) = 3 \text{ ms}^2$

Se pide: Calcular la varianza total del sistema $Var(T)$ para configurar las alertas del monitor de rendimiento.

1

Varianza de una suma de variables

Dado que $X$ e $Y$ no son independientes (su covarianza no es cero), usamos la fórmula general de la varianza para combinaciones lineales:

$$Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y)$$
2

Sustitución de valores

Insertamos las métricas conocidas del sistema:

$$Var(T) = 4 + 9 + 2(3)$$ $$Var(T) = 13 + 6 = 19$$
Resultado Problema 2
$$Var(T) = 19 \text{ ms}^2$$
💡 Interpretación: Si las variables fueran independientes, la varianza total sería solo 13 ms². Sin embargo, como el tráfico alto satura simultáneamente la VM y la base de datos (covarianza positiva), la inestabilidad total de la latencia sube hasta 19 ms².
